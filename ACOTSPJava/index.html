ACOTSPJava

A Java implementation of ACO algorithms for the TSP

Source code

The source code is available at
https://github.com/adibaba/ACOTSPJava

You have to include the Apache Commons CLI library 1.2 http://commons.apache.org/proper/commons-cli/

Project information

ACOTSPJava is based on the ACOTSP project of Thomas Stuetzle:
ACO algorithms for the TSP
http://iridia.ulb.ac.be/~mdorigo/ACO/aco-code/public-software.html
